# LM Profesionales - Proyecto Django

## Descripción
Aplicación web desarrollada en Django que permite gestionar profesionales, servicios y entregables.

## Funcionalidades

- Crear Profesionales
- Crear Servicios
- Crear Entregables
- Buscar Servicios por nombre

## Cómo probar

1. Ejecutar el servidor:
   python manage.py runserver

2. Ir a:
   http://127.0.0.1:8000/

3. Navegar por:
   - Profesionales
   - Servicios
   - Entregables
   - Buscar

## Tecnologías
- Python
- Django
